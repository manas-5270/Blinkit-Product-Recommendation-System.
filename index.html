About Blinkit AI System

The **Blinkit Product Recommendation System** is built on top of regression and classification models to suggest optimal grocery products based on consumer settings.

Using a **Random Forest Regressor** trained on historical customer preferences, ratings, price sensitivities, and brands, the model predicts the likelihood of purchase (the prediction index). A secondary **Decision Tree Classifier** maps product associations within categories.

Key Tech Stack & Model Features

  • StandardScaler & LabelEncoder preprocessors for encoding brand and category strings.
  • Bento Grid Layout and HSL variables for an ultra-premium SaaS UI inspired by Arc Browser and Stripe.
  • Plotly / Chart.js charts showing distributions and dependencies dynamically.
  • Scikit-Learn (0.97 accuracy) Decision Tree classification for category matching.
